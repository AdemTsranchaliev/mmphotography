# MM Photography

Изчистено портфолио за фотограф — Next.js, топла палитра, пълноекранен герой.

## Старт

```bash
npm install
npm run dev
```

Отвори [http://localhost:3000](http://localhost:3000).

## Стек

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- `next/image` + Unsplash placeholder снимки
